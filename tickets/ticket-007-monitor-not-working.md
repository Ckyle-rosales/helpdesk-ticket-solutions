# Ticket 007: Monitor Not Displaying Output

## Issue
User reports that their monitor is not displaying any image after powering on their computer.

## Environment
- Windows 10 desktop
- External monitor connected via HDMI/DisplayPort
- Dedicated graphics card installed

## Troubleshooting Steps
1. Verified that the monitor was powered on
2. Checked that the display cable was securely connected to both the monitor and computer
3. Confirmed the monitor was set to the correct input source
4. Tested the monitor with a different cable
5. Connected the monitor to another device to verify functionality
6. Restarted the computer
7. Reseated the graphics card and confirmed GPU power connections were secure
8. Tested a different display output port on the graphics card

## Findings
The monitor was functional, but the display cable was connected to the incorrect port and the monitor input source was not set properly.

## Resolution
Connected the cable to the correct graphics card output and selected the proper input source on the monitor. Display output was restored successfully.

## Key Takeaways
Display issues can be caused by incorrect cable placement, wrong monitor input selection, faulty cables, or loose GPU connections. Verifying physical connections first helps isolate the issue quickly.

## Customer Communication
Explained the issue in simple terms and showed the user how to verify the monitor input source and cable connection if the issue happens again.
