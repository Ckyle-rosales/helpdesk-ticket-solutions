# Ticket 003: Slow Internet Performance

## Issue
User reports that their internet connection is significantly slower than usual, with delays when browsing and downloading files.

## Environment
- Windows 10 laptop
- Home WiFi network
- Multiple devices connected

## Troubleshooting Steps
1. Ran a speed test to confirm reduced internet performance
2. Checked if multiple devices were consuming bandwidth on the network
3. Restarted the router to clear potential network congestion
4. Verified signal strength and distance from the wireless access point
5. Checked for interference from nearby devices and networks
6. Ran a virus and malware scan to rule out background activity
7. Disabled unnecessary background applications using Task Manager

## Findings
Network congestion and wireless interference were causing reduced speeds and increased latency.

## Resolution
Restarted the router and reduced interference by moving closer to the access point. Improved performance was confirmed with a follow-up speed test.

## Key Takeaways
Slow internet can often be caused by network congestion, interference, or excessive background usage rather than ISP issues.

## Customer Communication
Explained the cause of the slow speeds and advised the user on optimizing device placement and limiting background usage for better performance.
