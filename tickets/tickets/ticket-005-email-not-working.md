# Ticket 005: Email Client Not Sending or Receiving Messages

## Issue
User reports that their email client is not sending or receiving messages.

## Environment
- Windows 10 laptop
- Email client (Outlook/Gmail app)
- Home network

## Troubleshooting Steps
1. Verified internet connectivity by opening a web browser
2. Checked email login credentials for accuracy
3. Reviewed email server settings (SMTP/IMAP/POP3)
4. Confirmed correct ports and encryption settings
5. Disabled firewall temporarily to test connectivity
6. Checked for application updates
7. Attempted to access email via web browser to isolate issue

## Findings
Incorrect SMTP server settings were preventing outgoing email transmission.

## Resolution
Updated the email client settings with the correct SMTP server and port configuration. Email functionality was restored.

## Key Takeaways
Email issues are often caused by incorrect server configurations or authentication errors. Verifying settings and isolating the issue using web access helps identify the root cause.

## Customer Communication
Explained the issue in simple terms and guided the user on verifying their email settings for future troubleshooting.
